# Event Management System

Project UTS Enterprise Application Integration (EAI)  
Menggunakan arsitektur Microservices.

## Services

- User Service
- Event Service
- Registration Service
- Notification Service

## Event Service

Fitur:

- Create Event
- Get All Events
- Get Event by ID
- Update Event
- Delete Event
- Update Quota

## Teknologi

- Laravel
- MySQL
- REST API
- Postman
